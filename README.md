<h1 align="center">SynthStack</h1> 

---

<div align="center">
<p>
SynthStack is a VST plugin manager. On the surface, it sounds simple. However, structure determines whether it becomes a mess of spaghetti code or a clean, professional system (which you can actually learn from). I'm using a Modular Monolith. That means one deployable application - but divided into clearly separated modules that behave as if they were independent. No network calls between modules. No microservices complexity. Just logical boundaries enforced by code discipline.</p></div>
